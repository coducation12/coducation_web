# Coducation 변경 이력

## [0.1.0] - 2024-12-19

### ✨ 추가된 기능

#### 인증 시스템
- 하이브리드 인증 시스템 구현
  - 학생/학부모: 데이터베이스 직접 인증
  - 강사/관리자: Supabase Auth 통합
- 비밀번호 입력 필수화 (개발용 자동 로그인 제거)
- 역할 기반 접근 제어 (RLS)

#### 대시보드
- 역할별 대시보드 구현
  - 학생 대시보드: 학습 진행, 타자 연습, 출석 체크
  - 학부모 대시보드: 자녀 현황, 출석률, 학원비
  - 강사 대시보드: 담당 학생 관리, 커리큘럼 관리
  - 관리자 대시보드: 전체 시스템 관리

#### 커리큘럼 관리
- 커리큘럼 생성 및 수정 기능
- 단계별 체크리스트 관리
- 결과물 업로드 (이미지/URL/파일)
- 메모 및 피드백 시스템
- 완료 처리 기능

#### 타자 연습 시스템
- 기본 타자 연습 (한글/영어)
  - 자리연습
  - 실전연습
- 단어 타자 연습
- AI 기반 타자 연습 문제 생성 (Google AI Genkit)
- 타자 속도 및 정확도 측정
- 결과 저장 및 통계

#### 출석 관리
- 출석 일정 설정 (요일별, 시간별)
- 출석 체크 기능
- 출석률 계산 및 통계
- 출석 캘린더

#### 커뮤니티
- 게시글 작성/수정/삭제
- 댓글 시스템
- 이미지 업로드 (다중)
- 역할 배지 표시
- Soft Delete 구현

#### 강사진 관리
- 강사 프로필 관리
- 메인 페이지 강사진 섹션
- DB 기반 자동 정렬 (원장/부원장 우선)
- 강사 상세 모달
- 반응형 레이아웃 (2+3 구조)

#### 이미지 관리
- 이미지 자동 압축 (WebP 변환)
- 자동 리사이징 (최대 1920x1080)
- 프로그레시브 로딩
- 1년 캐시 설정

#### 랜딩 페이지
- 히어로 섹션
- 학원 소개 섹션
- 강사진 섹션
- 커리큘럼 섹션
