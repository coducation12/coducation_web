# Coducation 프로젝트 문서

이 디렉토리에는 Coducation 프로젝트의 모든 문서가 포함되어 있습니다.

## 📚 문서 목록

### 1. [프로젝트 개요](./PROJECT_OVERVIEW.md)
프로젝트의 전체적인 소개, 목표, 현황, 구조를 설명합니다.

### 2. [아키텍처](./ARCHITECTURE.md)
시스템 아키텍처, 인증 시스템, 데이터베이스 구조, 프론트엔드 아키텍처를 상세히 설명합니다.

### 3. [기능 상세](./FEATURES.md)
각 기능별 상세 설명, 사용 방법, 컴포넌트 정보를 제공합니다.

### 4. [데이터베이스 스키마](./DATABASE_SCHEMA.md)
데이터베이스 테이블 구조, RLS 정책, 주요 쿼리 예시를 설명합니다.

### 5. [개발 가이드](./DEVELOPMENT.md)
개발 환경 설정, 개발 워크플로우, 코딩 가이드라인을 제공합니다.

### 6. [변경 이력](./CHANGELOG.md)
프로젝트의 버전별 변경 사항을 기록합니다.

## 🚀 빠른 시작

### 새 개발자를 위한 가이드

1. [프로젝트 개요](./PROJECT_OVERVIEW.md)를 읽어 프로젝트를 이해하세요.
2. [개발 가이드](./DEVELOPMENT.md)를 따라 개발 환경을 설정하세요.
3. [아키텍처](./ARCHITECTURE.md)를 읽어 시스템 구조를 파악하세요.
4. [기능 상세](./FEATURES.md)를 참고하여 기능을 이해하세요.

### 기존 개발자를 위한 가이드

- 새로운 기능 추가: [기능 상세](./FEATURES.md) 참고
- 데이터베이스 작업: [데이터베이스 스키마](./DATABASE_SCHEMA.md) 참고
- 배포: [배포 가이드](../DEPLOYMENT_GUIDE.md) 참고
