# 학생 진도 및 성과 관리 시스템 (Student Progress System)

학생의 학습 여정을 기록하고 학부모와 공유하며, 커뮤니티 성장을 도모하는 통합 관리 시스템을 위한 공식 문서입니다.

---

## 1. 개요 및 목적
- **목적**: 강사가 학생의 학습 과정과 성과를 체계적으로 기록하고, 학생/학부모는 대시보드를 통해 이를 한눈에 확인하여 학습 동기를 부여받음.
- **핵심 가치**: 데이터 기반의 학습 상담, 포트폴리오 누적 관리, 커뮤니티 내 성과 공유.

---

## 2. 데이터베이스 설계 (Database Schema)

모든 학생별 성과 데이터는 **`students`** 테이블 내에 **JSONB** 타입으로 통합 관리하여 유연한 데이터 확장을 지원합니다.

### 2.1 JSONB 데이터 구조 상세

#### A. `learning_progress` (누적 수업 결과물)
- **용도**: 과정별 상태(진행/완료)와 그에 따른 포트폴리오(이미지, URL) 누적 저장.
- **구조**: 
  ```json
  [
    {
      "curriculum_id": "uuid",          // 연동된 커리큘럼 ID
      "status": "ongoing | completed",  // 과정 완료 여부
      "results": [                      // 누적 타임라인 (Portfolio)
        {
          "id": "item-uuid",
          "date": "2024-03-20",
          "title": "나만의 파이썬 웹",
          "description": "짧은 결과물 설명",
          "imageUrl": "https://...",      // 스토리지 저장 경로
          "url": "https://...",           // 프로젝트 외부 링크
          "isShared": true               // 커뮤니티 공유 여부
        }
      ]
    }
  ]
  ```

#### B. `achievement_records` (자격증 및 수상 내역)
- **용도**: 취득 완료된 성과와 향후 목표 자격증 관리.
- **구성**: 
  - **카테고리 분류**: ITQ, COS Pro, GTQ, DIAT, Topcit, 정보처리기능사 등.
- **구조**: 
  ```json
  {
    "attained": [ // 실제 취득한 내역
      { "type": "award | cert", "title": "...", "date": "...", "category": "..." }
    ],
    "targets": [ // 도전 중인 목표 (분류되어 선택 가능)
      { "category": "ITQ", "title": "엑셀 A등급", "isAttained": false, "targetDate": "..." }
    ]
  }
  ```

#### C. `typing_stats` (타자 성적)
- **용도**: 한글/영어 별 최고 타수 기록.
- **구조**: `{ "ko": { "maxSpeed": 450, "date": "..." }, "en": { "maxSpeed": 380, "date": "..." } }`

## 3. 사용자 그룹별 기능 및 UI 사양

### 3.1 강사 계정 (Teacher Workspace)
1. **과목 매칭 및 진도 관리**: `main_subject`, `sub_subject`를 활용해 수강 중인 과정을 지정하고 상태(진행/완료) 변경.
2. **진도 관리 모달 (StudentProgressModal)**:
   - 결과물 이미지 업로드/URL 입력 및 누적 저장.
   - 수상/자격증 내역 기록 및 **분류된 목록**에서 목표 자격증 선택.
   - **[ ] 커뮤니티 공유** 체크 시 `community_posts`에 자동 등록.
3. **접근성**: 대시보드 및 학생 관리 페이지에서 아이콘(📈) 클릭 시 즉시 모달 출력.

### 3.2 학생 및 학부모 (Dashboard)
1. **수업 내역 카드(Course Cards)**:
   - 진행 중인 수업: 상단 배치 및 '학습 중' 배지 노출.
   - 완료된 수업: 하단 배치 및 '완료' 배지 노출.
2. **성과 갤러리 및 모달**:
   - 카드 클릭 시 해당 과정의 결과물 타임라인이 리스트 형태로 노출.
   - 결과물 클릭 시 **고해상도 모달 뷰**로 이미지와 상세 링크 확인.
3. **업적 타임라인**: 취득 자격증 및 수상 내역을 배지 또는 목록 형태로 조회.

---

## 4. 핵심 데이터 이관 계획 (XP 및 연동)
- **XP 데이터 이관**: 기존 **`users`** 테이블의 `total_xp` 칼럼을 **`students`** 테이블로 이동하여 학생 정보와 일치시킴.
- **커뮤니티 연동**: 공유 시 게시글 제목 형식: `[자랑하기] OOO 학생의 새로운 성과가 등록되었습니다!`
